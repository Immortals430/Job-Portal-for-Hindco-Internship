## Run Locally

Clone the project

```bash
  git clone https://github.com/Immortals430/Job-Portal-for-Hindco-Internship.git
```

Go to the project directory

```bash
  cd Job-Portal-for-Hindco-Internship
```


Install dependencies

```bash
  npm install
```

Start app

```bash
  node run dev
```
